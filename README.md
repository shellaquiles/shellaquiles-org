## 🚀 Características

- **Landing Page** con diseño tipo terminal único
- **Sistema de Blog** integrado con routing SPA
- **Diseño Responsive** y accesible
- **Animaciones** suaves y efectos visuales
- **Tema Terminal** consistente en toda la aplicación
